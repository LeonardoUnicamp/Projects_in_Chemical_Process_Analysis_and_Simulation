# Projects in Chemical Process Analysis and Simulation (QUI042 - UTFPR-FB)

Repository link: https://github.com/LeonardoUnicamp/Projects_in_Chemical_Process_Analysis_and_Simulation

## Challenge 1
Replicate the methodology from the article "Klassische Runge-Kutta-Formeln fünfter und siebenter Ordnung mit Schrittweiten-Kontrolle" by E. Fehlberg (DOI: https://doi.org/10.1007/BF02234758) using Python.

## Challenge 2
Replicate the methodology from the article "Thermal denaturation: is solid-state fermentation really a good technology for the production of enzymes?" by Marcelo Muller dos Santos, Alexandre Souza da Rosa, Silvia Dal’Boit, David A. Mitchell, and Nadia Krieger (DOI: https://doi.org/10.1016/j.biortech.2003.11.007) using Python.
